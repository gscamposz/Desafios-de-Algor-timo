Desafios em Java – Switch Case e Estruturas de Repetição

Este repositório contém dois desafios desenvolvidos em Java com o objetivo de praticar conceitos fundamentais da linguagem, como estruturas condicionais, estruturas de repetição, entrada de dados pelo usuário e lógica de programação.

📌 Desafio 1 – Calculadora de Áreas Geométricas (Switch Case)

Sistema interativo que permite ao usuário calcular a área de diferentes figuras geométricas através de menus utilizando a estrutura switch case.

Funcionalidades:

* Cálculo da área de:
    * Quadrado
    * Retângulo
    * Trapézio
    * Triângulo Equilátero
    * Triângulo Isósceles
    * Triângulo Escaleno
* Menus hierárquicos para seleção das figuras.
* Repetição contínua até que o usuário escolha encerrar o programa.
* Utilização de fórmulas matemáticas e da classe Math.

Conceitos praticados:

* switch case
* Estruturas de repetição (while)
* Entrada de dados com Scanner
* Operações matemáticas
* Organização de menus

🎮 Desafio 2 – Movimentação de Personagem em Matriz (For)

Programa que gera um tabuleiro personalizado utilizando caracteres definidos pelo usuário e permite movimentar um personagem pelo cenário através do teclado.

Funcionalidades:

* Definição do tamanho do mapa (linhas e colunas).
* Escolha de dois caracteres para formar o cenário.
* Posicionamento inicial do personagem.
* Movimentação utilizando:
    * W → Cima
    * A → Esquerda
    * S → Baixo
    * D → Direita
* Encerramento do programa com a tecla Q.
* Atualização do mapa a cada movimento.

Conceitos praticados:

* Estruturas de repetição (for e while)
* Matrizes lógicas
* Controle de posição
* Manipulação de Strings
* Estruturas condicionais
* Desenvolvimento de lógica para jogos simples em terminal
